RestauLb is a modern and interactive web application designed to help users discover some of the best restaurants across Lebanon and reserve tables easily. The platform provides a clean and responsive user interface where visitors can explore featured restaurants, search by name or location, and view restaurant locations on an interactive map powered by Leaflet. Users can also filter restaurants by region and quickly make reservations through a WhatsApp integration that simplifies the booking process. The project focuses on combining modern web design with practical functionality to create a smooth and engaging user experience while showcasing front-end development skills using HTML, CSS, JavaScript, and TailwindCSS.
